# Rural Community App (Frontend)

## Setup

1. Clone the repo
2. Run `npm install`
3. Set API URL in `.env`
4. Run `npm start`

## Features

- Homepage with services, products, news, contact form
- User signup/login
- User dashboard with bookings
- Responsive design
